# 용어 사전

- ProcessorNodeStatus: 처리기 노드 상태
- ProcessorNode: 처리기 노드
    - node_a
    - node_b
    _ ...
- ProcessorType: 처리기 유형
    - 렌더러
    - 형식 변환기
    - 모델 처리기
- Job: 작업
- JobExecution: 작업 실행
- JobExecutionStatus: 작업 실행 상태
- Asset: 에셋
- AssetTag: 에셋 태그
- AssetFormat: 에셋 형식
    - Wavefront OBJ
    - glTF
    - STL
    - DELTA
    - ...
- AssetType: 에셋 유형
    - 3D 모델
    - 이미지
- EncryptionKey: 암호화 키
